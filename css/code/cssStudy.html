한국행정연구원에서는 2022년 연구사업의 일환으로 ｢공공리더십 역량모델 개선 연구｣ 와 ｢공공리더십 역량모델 진단도구 개발 연구｣를 수행 중입니다. 본 연구과제는 변화하는 행정 환경에 빠르고, 효과적으로 대응하기 위해 과장급 공공리더들에게 필요한 역량의 선정과 함께, 과장급 공공리더십 역량을 진단하고자 합니다. 본 진단도구는 평가 용도가 아닌 개인의 강점과 약점을 파악하여 보완할 수 있도록 하는 것을 목적으로 하고 있습니다. 본 설문 실행 후 타당도 및 신뢰도가 확보되면 향후 실제 과장급 공공 리더십 역량을 측정하는 도구로 활용될 예정입니다.
  답변 내용은 "통계법 제33조(비밀의 보호)에 따라 익명성이 보장되며, 연구 이외의 목적으로 사용되지 않고, 연구 완료 시 바로 폐기됩니다. 설문 응답에 소요되는 시간은 10-15분 정도이며, 귀하의 소중한 의견이 새로운 진단도구에 반영될 수 있도록 노력하겠습니다.
  바쁘신 중에도 귀한 시간을 내셔서 정성껏 응답해주심에 다시 한 번 깊은 감사를 드립니다.
